# Clube de Desbravadores Falcão Negro

Site oficial do Clube de Desbravadores **Falcão Negro** — Cascavel, Paraná.

> "Mas os que esperam no SENHOR renovarão as suas forças; subirão com asas como águias." — Isaías 40:31

---

## Sobre o Projeto

Site desenvolvido em **Astro** com foco em performance, SEO e design fiel à identidade visual do clube. O objetivo é atrair novos membros e apresentar o clube para pais e jovens de Cascavel.

## Tecnologias

- [Astro](https://astro.build/) — framework web estático com excelente SEO
- CSS puro com variáveis e animações
- Schema.org para dados estruturados
- Open Graph para compartilhamento em redes sociais

## Seções do Site

- **Hero** — apresentação animada com o falcão e versículo âncora
- **Sobre** — história e estatísticas do clube
- **Pilares** — Visão de Propósito, Força Inabalável e Voo de Esperança
- **Atividades** — acampamentos, honras, banda, projetos comunitários e mais
- **Classes Progressivas** — as 6 classes com cores e nomes corretos
- **Contato** — WhatsApp, Instagram e card de inscrição

## Como Rodar Localmente

```bash
# Instalar dependências
npm install

# Iniciar servidor de desenvolvimento
npm run dev

# Acessar em http://localhost:4321
```

## Build para Produção

```bash
npm run build
npm run preview
```

## Deploy

O site está configurado para deploy automático na **Vercel**. Qualquer push na branch `main` dispara um novo deploy.
